This is a Machine Learning App for predicting whether a breast tumour is MALIGNANT or BENIGN.

The idea of the project is to reduce invasive procedures in the diagnosis of Breast Cancer and other related diseases. The Breast Predictor App will is better used by medical or health care professionals than for domestic use as some measurements involved would require some expertise.

The App is an AI application trained using Support Vector Machine and GridSearch CV. The data-set trained was a copy of UCI ML Breast Cancer Wisconsin (Diagnostic) datasets. of more than 500 data points and 30 features out of which 8 most important features were selected for easy use.The exploratory data analysis can be found in this notebook and the codes can be found in this repository

The user is expected to input some values (in millimeters) as instructed based on the observed properties of the tumour which help the machine to predict the kind of tumour.

The model can be improved by getting better data sets which could produce more accurate results and could be specific to localities.

Note for user: Make sure you input numbers (could have decimals or not) and the values should be within the range of the values provided in the input.
